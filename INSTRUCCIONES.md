# 🐾 Amigos Peludos - Frontend

Este es el frontend del proyecto **Amigos Peludos**, desarrollado con React + Vite.

## 🛠️ Tecnologías

- React
- Vite (con JavaScript + SWC)
- HTML, CSS
- Node.js / npm

## 🚀 Instrucciones para correr el proyecto localmente
instalar dependencias
-npm install

## Levantar proyecto 
-npm run dev
